# Secure Password Generator (OOP)

A simple, robust Python application that generates secure, customizable passwords using Object-Oriented Programming principles.

## ✨ Features
- Custom length selection.
- Toggle for Uppercase, Numbers, and Special characters.
- Built with **Python OOP** for modularity and reusability.

## 🚀 How to Run
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/password-generator.git`
2. Navigate to folder: `cd password-generator`
3. Run the app: `python main.py`

## 🛠️ Concepts Used
- **Encapsulation**: Logical grouping of data and methods within the `PasswordGenerator` class.
- **Private Methods**: Using `_build_char_pool` to handle internal logic.
